Font Files for MAUI Application
================================

This folder should contain the following font files:
- OpenSans-Regular.ttf
- OpenSans-Semibold.ttf

You can download Open Sans from Google Fonts:
https://fonts.google.com/specimen/Open+Sans

After downloading:
1. Extract the TTF files
2. Place them in this folder
3. The project file is already configured to include them

Alternatively, you can use any other fonts by:
1. Adding the TTF files to this folder
2. Updating MauiProgram.cs ConfigureFonts() method
3. Updating Resources/Styles/Styles.xaml FontFamily references
