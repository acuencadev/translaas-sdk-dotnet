# Translaas.Caching.File

**File-based** and **offline** caching for the Translaas SDK: persist translation data locally (for example JSON on disk) and sync with the API when online.

## Installation

```powershell
dotnet add package Translaas.Caching.File
```

Brings in **Translaas.Client**, **Translaas.Caching**, and **Translaas.Models**.

## When to use this package

- Desktop or mobile apps that must work with limited connectivity.
- Hybrid **memory + file** cache scenarios.
