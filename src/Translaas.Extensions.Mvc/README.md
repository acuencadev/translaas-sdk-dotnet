# Translaas.Extensions.Mvc

**ASP.NET Core MVC / Razor** integration: tag helpers and helpers for using Translaas translations in `.cshtml` views.

## Installation

```powershell
dotnet add package Translaas.Extensions.Mvc
```

Depends on **Translaas.Extensions.DependencyInjection** (and the rest of the stack transitively). Target **netstandard2.0** for older ASP.NET Core, or **net8.0** / **net10.0** for framework reference scenarios.

## When to use this package

- Razor Pages or MVC apps where you want declarative translation usage in views.
