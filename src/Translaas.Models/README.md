# Translaas.Models

Data transfer objects (DTOs) for the **Translaas Translation Delivery API** .NET SDK: request and response models, request context, and error types.

## Installation

```powershell
dotnet add package Translaas.Models
```

## When to use this package

- You reference Translaas types in your own libraries or tests without pulling in the HTTP client.
- You build a custom integration on top of the API contract.

Most applications should install **Translaas.Client** or **Translaas.Extensions.DependencyInjection** instead; they bring in **Translaas.Models** transitively.
